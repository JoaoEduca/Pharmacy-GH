# Drogaria GH — Projeto Faculdade

## Estrutura do Projeto

```
FarmaciaGH/
  frontend/
    css/
      styles.css        → estilos
    js/
      firebase.js       → conexão com o Firebase
      produtos.js       → busca produtos do Firestore
    img/                → imagens dos produtos
    index.html          → página inicial
    produtos.html       → página de produtos (conectada ao banco)
    carrinho.html       → página do carrinho
    login.html          → página de login
    cadastro.html       → cadastro de usuarios
    feedback.html       → página de feedback
```



## Stack

- **Frontend:** HTML, CSS, Bootstrap 5
- **Banco de dados:** Firebase Firestore
- **Hospedagem:** Firebase Hosting (Visando)

## Funcionalidades

- Listagem de produtos buscados do Firestore em tempo real
- Filtro por categoria (Medicamentos, Cabelo, Bebê, Beleza, Conveniência)
- Busca por nome de produto
